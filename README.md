# KrushiDhan

A smart agriculture platform designed to empower Indian farmers with technology-driven solutions. KrushiDhan provides a unified dashboard for market prices, weather forecasts, government schemes, crop disease detection, equipment rentals, yield selling, and more.

## Features

- **Market Prices** - Real-time mandi prices
- **Weather Forecasting** - Location-based weather data
- **Government Schemes** - Agricultural schemes and subsidies
- **Crop Disease Detection** - Image-based identification
- **Equipment Rental** - Rent agricultural equipment
- **Yield Marketplace** - Sell crops directly
- **Fertilizer Dealers** - Find nearby dealers
- **Cold Storage Locator** - Interactive map
- **Agricultural Loans** - Browse loan options
- **News Feed** - Agriculture news

## Tech Stack

### Frontend
- React 19, React Router v7, Tailwind CSS, Leaflet, Axios, React Toastify

### Backend
- Node.js, Express.js, MongoDB, Mongoose, Multer, JWT

## Getting Started

### Backend Setup
```bash
cd backend
cp .env.example .env
npm install
npm start
```

### Frontend Setup
```bash
cd Frontend
npm install
npm start
```
